Assignment 03: Recursive and Non-Recursive Fibonacci Sequence

The Fibonacci sequence is the series of numbers where each number is the sum of the two previous numbers - 0, 1, 1, 2, 3, 5, 8, 13, and so on.

This page contains a JavaScript function that gives you the value based on running the calculation X number of times; for example calling fibonacci(4) gives you the answer of 5, an argument of 5 gives an answer of 8, etc.

You must add a new function named fibonacciRecursive() that implements the same functionality in a recursive manner - one where the function calls itself - and modify the README.txt file to explain your code.
